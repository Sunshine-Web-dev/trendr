== Description ==
json API for BuddyPress. This plugin creates json api endpoints for https://github.com/TRM-API

== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `TRS-API` and find this plugin in the results
3. Click 'Install'
4. Once installed, activate via the 'Plugins -> Installed' page


= Manual Installation =

1. Upload `trs-api` to the `/trm-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
