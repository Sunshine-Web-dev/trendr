== Description ==
Show number of profile views count by other members and recent visitors of member profile. And also show who is visiting the perticual member most.
 Any queries please send emails to gopal@dhrusya.com.

== Installation ==
installing and using trendr-profile-visitors is easy:

* Upload /trendr-profile-visitors/ to the /trm-content/plugins/ directory
* Activate the plugin through the \'Plugins\' menu in Trnder
* Done. Now you can see profile views and recent visitors on members account page.
