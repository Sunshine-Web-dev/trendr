--- trendr Default ---
Clean and stylish, trendr Default lets you build a social network straight out of the box. Make it yours with a custom menu, header image, and background. Along with five widgetized areas (one in the sidebar, four in the footer), TRS-Default supports featured images (as custom header images on posts and pages) and is furnished with an optional one-column page template that removes the sidebar, and a stylesheet for the admin Visual Editor.


--- Installation ---
The theme is bundled with trendr. After activating the plugin, trendr Default will be added to the "Appearance > Themes" menu in your WordPress admin area.


--- About trendr ---
Social networking in a box. Build a social network for your company, school, sports team or niche community all based on the power and flexibility of WordPress. trendr will let users register on your site and start creating profiles, posting messages, making connections, creating and interacting in groups and much more.

For help with trendr Default, or for more information about trendr, please visit http://trendr.org/.


--- Building a trendr theme ---
If you want to make a custom theme based on trendr Default, DO NOT copy and edit it. By doing this you will make updates and maintenance much harder for yourself. Instead, please review this codex page for instructions on how to build a trendr child theme:

http://codex.trendr.org/how-to-guides/building-a-trendr-child-theme/
